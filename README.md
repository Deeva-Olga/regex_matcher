### Движок регулярных выражений

Напишите приложение, которое будет повторять логику regex_matcher из папки example/  
Способ запуска показан в run.sh.

В корне вашего репозитория должен быть скрипт package.sh, который собирает test_bins.tar.gz (в корне), который будет распакован на проверяющей машине в /opt/test_bins

**Внимание!**  
regex_matcher - это не бейзлайн, это просто иллюстрация того, что необходимо сделать
